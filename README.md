# Student_Performance_Predictive_Model
Machine learning model to predict student pass/fail using GPA, attendance, and study habits with Decision Tree, Logistic Regression, and ROC analysis.
# Student Performance Predictive Model

## Objective
Predict whether a student will pass or fail using academic and demographic information.

## Dataset Features
- GPA
- Attendance Rate
- Study Hours Per Week
- Age
- Gender
- Major
- Part-time Job
- Extra Curricular Activities

## Methods
- Data Cleaning & Preprocessing
- Label Encoding for categorical variables
- Decision Tree Classifier
- Logistic Regression

## Evaluation Metrics
- Accuracy
- Classification Report
- Confusion Matrix
- ROC Curve (AUC)
- Feature Importance

## Result
GPA and Attendance Rate were found to be the most important predictors of student success.

## Tools Used
Python, scikit-learn, pandas, matplotlib, Google Colab
